🐾 Хатинка Лапок
Вебзастосунок для пошуку та прилаштування домашніх тварин.

📖 Про проєкт
«Хатинка Лапок» — це вебсайт, який допомагає користувачам знайти собі пухнастого друга.
На сайті можна переглядати доступних тварин, дізнаватися детальну інформацію про них та залишати заявку,
 щоб взяти тварину додому.

✨ Основні можливості
Перегляд списку домашніх улюбленців.
Пошук та фільтрація домашніх улюбленців.
Детальна інформація про кожну тварину.
Форма для оформлення заявки.
FAQ секція з відповідями на поширені запитання.
Історії успішного прилаштування тварин.
Адаптивний дизайн для різних пристроїв.
